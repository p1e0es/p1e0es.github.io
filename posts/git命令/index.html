Git命令

2024-05-24

常用命令

将当前目录初始化为git可管理的仓库

git init

查看所有分支

git branch -a

在本地创建并切换到分支main

git checkout -b main

删除本地分支main

git branch -d main

删除远程分支master

git push origin :master

将文件添加在本地暂存区

git add .

将文件提交到本地仓库

git commit -m "Just Test"

关联远程仓库

git remote add origin 你的远程库地址

查看远程仓库的地址

git remote -v

修改远程仓库的地址

git remote set-url origin 新的仓库地址

将远程仓库的main分支和本地项目同步合并

git pull --rebase origin main

将本地项目推送到远程仓库的main分支

git push -u origin main

查看状态

git status

新项目提交

先在github新建仓库 在本地项目目录中，使用git进行初始化

git init

配置git的用户名和邮箱

# 全局配置
git config --global user.name "username"
git config --global user.email "email"

# 本项目配置
git config user.email "email"
git config user.name "username"

查看配置

git config --list

设置远程仓库地址

git remote add origin "url"

查看远程仓库地址

git remote -v

在本地创建并切换到main分支

git checkout -b main

将远程仓库的main分支和本地项目同步合并

git pull --rebase origin main

将文件添加到暂存区

git add .

将文件提交到本地仓库

git commit -m "commit and so on"

将文件推送到远程仓库的main分支

git push -u origin main
# 输入github用户名
# 输入github的PAT凭证，2021.8.13密码验证不再支持

存储凭证

git config --global credential.helper cache

删除凭证

git config --global --unset credential.helper

项目修改后提交

项目更新后，将更新的文件添加到暂存库

git add -A

提交到本地仓库

git commit -m "update"

将项目推送到远程仓库

git push
